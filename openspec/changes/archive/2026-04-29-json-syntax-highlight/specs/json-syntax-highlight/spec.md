## ADDED Requirements

### Requirement: JSON syntax highlighting
The raw JSON toggle SHALL display JSON with color-coded syntax: keys in blue, string values in dark blue, numbers in orange, booleans in purple, null in gray.

#### Scenario: Syntax-highlighted JSON
- **WHEN** user expands "查看原始 JSON"
- **THEN** the JSON is displayed with colored keys/values in a fixed-width font on a light gray background
